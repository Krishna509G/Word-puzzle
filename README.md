# Word-puzzle
A fullscreen, competitive multiplayer word-guessing game built in Python using Tkinter. Players take turns guessing letters, earning points, using hints, and solving words across 10 exciting rounds. The game includes categories, custom-word mode, animations, scoring, a final leaderboard, and a polished UI.
